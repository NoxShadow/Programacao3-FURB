Questão 5: Crie um formulário em HTML contendo 3 campos do tipo input text, sendo um campo o número 1, outro campo o número 2 e por fim o número 3. Coloque labels para identificar cada um dos campos. Alinhe esses campos um ao lado do outro de modo que se a resolução do navegador for muito grande, o campo se posicione abaixo, tornando assim uma tela responsiva. Edite o CSS dos componentes do formulário de modo que não fiquem originais (HTML puro). Crie uma função que lê esses valores numéricos provenientes dos elementos HTML existentes na página e abra um alerta com o valor da soma de todos eles, informando se o valor é par ou ímpar. Chame a função através do botão Submit do formulário.
